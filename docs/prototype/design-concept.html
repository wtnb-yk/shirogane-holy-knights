01 — Visual Direction

ビジュアルの方向性

ノエルの衣装カラーをそのままUIに落とし込んだ、モダンで機能的なツール。

衣装そのままの配色

クリームの温かい白、ドレスの紺、鎧のゴールドトリム、銀髪のグレー。ノエルのキャラデザから直接取った5色でUIを構成。

CreamNavyGold

モダンなツール

Linear / Notion のような機能的で洗練されたWebアプリの佇まい。ファンタジーUIではなく、実用的で日常的に使えるデザイン。

FunctionalClean

世界観は最小限

⚔アイコンとコピー（「団員カルテ」等）で騎士団を暗示。中世風の装飾やファンタジーフォントは一切使わない。

SubtleCopy Only
判断基準: UIのどの要素も、ノエルを知らない人が見ても「きれいなWebアプリ」と思えること。ファン要素はコンテンツとコピーライティングで表現し、UIデザイン自体はニュートラルに保つ。

Design

クリーム(#FDFCFA)を紙色に、紺(#2C3A4E)をテキストに使用。ブランドのゴールドは使わず、レシートの「紙に印字」感を優先。アプリ全体のカラーシステムの中に自然に溶け込む。

Data Source

配信一覧のLocalStorageデータを自動集計。カテゴリ別視聴数・推定時間、探索率(視聴数/1,419件)、推し歴(最古チェック日からの経過)。データ未登録時は手動入力モードにフォールバック。

Viral Flow

記録がある → レシート即生成 →「画像をダウンロード」ボタン → PNG保存 → Xに手動投稿（自分の言葉を添えられる）。レシート下部のURLが「やってみて」連鎖の起点。

Implementation

@vercel/og (Satori)でEdge Runtime上にSVG→PNG変換。画面表示用とOGP生成用で同じReactコンポーネントを共用。バーコードはCSS生成。
